## 🔑 กุญแจทำคะแนน

**ปรนัย 40 คะแนน** — ท่องให้แม่น:

| จำคู่นี้ให้ได้ | |
|---|---|
| PDPA → **สคส. (PDPC)** | ไซเบอร์ → **สกมช.** · ธุรกรรมฯ → **ETDA** |
| PDPA แจ้งละเมิด **72 ชม.** | log พ.ร.บ.คอม **90 วัน** |
| ลิขสิทธิ์ ไม่ต้องจด (ชีพ+50ปี) | เครื่องหมายการค้า ต้องจด (**10 ปี** ต่อได้) · สิทธิบัตรประดิษฐ์ **20 ปี** |
| แฮกระบบ = **ม.5** · ข่าวปลอม = **ม.14** | DDoS = **ม.10** · ภาพตัดต่อ = **ม.16** |
| EU AI Act **4 ระดับเสี่ยง** | ETDA **6 หลักจริยธรรม** |

**อัตนัย 20 คะแนน** — เขียนตามโครง **ข้อเท็จจริง → ข้อกฎหมาย(มาตรา) → วินิจฉัย → สรุป+มิติจริยธรรม**
> ⭐ ทุกข้อปิดด้วย **"มิติจริยธรรม"** (Trust / เคารพสิทธิ / รับผิดชอบต่อสังคม / ได้สัดส่วน) = จุดได้คะแนนเหนือคนอื่นในวิชานี้

**ลำดับความสำคัญ:** PDPA (20 คะแนนย่อย) > หัวข้ออื่น — ทุ่มเวลาให้ PDPA มากที่สุด
