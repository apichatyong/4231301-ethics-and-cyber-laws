# ส่วน A — พ.ร.บ.ว่าด้วยการกระทำความผิดเกี่ยวกับคอมพิวเตอร์

| รายการ | ข้อมูล |
|---|---|
| ฉบับแรก | พ.ศ. **2550** (บังคับใช้ 18 ก.ค. 2550) |
| แก้ไขเพิ่มเติม (ฉบับ 2) | พ.ศ. **2560** (บังคับใช้ 24 พ.ค. 2560) |
| นิยามสำคัญ | **มาตรา 3** — ระบบคอมพิวเตอร์, ข้อมูลคอมพิวเตอร์, ข้อมูลจราจรทางคอมพิวเตอร์, ผู้ให้บริการ |

## 🏛️ บริบทยุคสมัย & ผู้ออกกฎหมาย (อาจารย์เน้น)

| รายการ | ฉบับแรก 2550 | ฉบับที่ 2 (2560) |
|---|---|---|
| **ยุค/บริบท** | หลัง**รัฐประหาร 19 ก.ย. 2549** (โค่นรัฐบาลทักษิณ) ยุค **คมช.** | หลัง**รัฐประหาร 2557** ยุค **คสช.** |
| **รัฐบาล (นายกฯ)** | **พล.อ. สุรยุทธ์ จุลานนท์** | **พล.อ. ประยุทธ์ จันทร์โอชา** |

> 💡 **แพตเทิร์นจำง่าย:** พ.ร.บ.คอม **ทั้ง 2 ฉบับ คลอดจากสภาแต่งตั้ง (สนช.) ในยุคหลังรัฐประหาร** (2550 = หลัง รปห.49 / 2560 = หลัง รปห.57)
> ⚖️ **เหตุผลที่ต้องออกปี 2550:** ตอนนั้นไทย**ยังไม่มีกฎหมายเอาผิดอาชญากรรมคอมพิวเตอร์โดยตรง** (แฮก/เจาะระบบเอาผิดยาก) จึงต้องมีกฎหมายเฉพาะ

## ⭐ ฐานความผิดและโทษ (จำตารางนี้ = หัวใจการสอบ)

| มาตรา | ความผิด | โทษสูงสุด |
|---|---|---|
| **ม.5** | เข้าถึง **ระบบ** คอมพิวเตอร์ที่มีมาตรการป้องกันโดยมิชอบ | จำคุก ≤ 6 เดือน / ปรับ ≤ 10,000 |
| **ม.6** | ล่วงรู้ **มาตรการป้องกัน** แล้วนำไปเปิดเผย | จำคุก ≤ 1 ปี / ปรับ ≤ 20,000 |
| **ม.7** | เข้าถึง **ข้อมูล** คอมพิวเตอร์โดยมิชอบ | จำคุก ≤ 2 ปี / ปรับ ≤ 40,000 |
| **ม.8** | **ดักรับ** ข้อมูลระหว่างส่งโดยมิชอบ | จำคุก ≤ 3 ปี / ปรับ ≤ 60,000 |
| **ม.9** | **แก้ไข/ทำลาย/เปลี่ยนแปลง** ข้อมูลผู้อื่นโดยมิชอบ | จำคุก ≤ 5 ปี / ปรับ ≤ 100,000 |
| **ม.10** | **รบกวน/ขัดขวางระบบ** จนทำงานไม่ปกติ (เช่น DoS) | จำคุก ≤ 5 ปี / ปรับ ≤ 100,000 |
| **ม.11** | **สแปม** — ส่งข้อมูล/อีเมลปกปิดแหล่งที่มา รบกวนผู้รับ | ปรับ ≤ 100,000 (รบกวนรำคาญ ≤ 200,000) |
| **ม.12** | **เหตุฉกรรจ์/เพิ่มโทษ ของ ม.9 + ม.10** (ดูตาราง 3 ชั้นด้านล่าง) | สูงสุด **จำคุก 5–20 ปี / ปรับ ถึง 400,000** |
| **ม.13** | จำหน่าย/เผยแพร่ **ชุดคำสั่ง (มัลแวร์)** เพื่อใช้กระทำผิด | จำคุก ≤ 1 ปี (เพื่อ ม.12 → ≤ 2 ปี) |
| **ม.14** | นำเข้า **ข้อมูลเท็จ/ปลอม/ลามก** กระทบบุคคล ฉ้อโกง ความมั่นคง ตื่นตระหนก | จำคุก ≤ 5 ปี / ปรับ ≤ 100,000 |
| **ม.15** | **ผู้ให้บริการ (ตัวกลาง)** ยินยอม/รู้เห็นกับความผิด ม.14 | รับโทษเท่าผู้โพสต์ (ดูข้อยกเว้น) |
| **ม.16** | นำเข้า **ภาพตัดต่อ/ดัดแปลง** ทำให้ผู้อื่นเสียชื่อเสียง/อับอาย | จำคุก ≤ 3 ปี / ปรับ ≤ 200,000 |

## 🔥 เจาะ ม.12 — เหตุฉกรรจ์/เพิ่มโทษ (3 ชั้น)

> **ม.12 ไม่ใช่ฐานความผิดใหม่** แต่เป็นการ **ยกระดับโทษของ ม.9 (แก้ไข/ทำลายข้อมูล) และ ม.10 (รบกวนระบบ/DoS)** เมื่อผลร้ายแรงขึ้น — เกาะอยู่กับ 2 มาตรานี้เท่านั้น

| ชั้น | เงื่อนไข | โทษ |
|---|---|---|
| **ม.12(1)** | ทำผิด ม.9/ม.10 แล้ว **ก่อความเสียหายแก่ประชาชน** (เกิดทันทีหรือภายหลังก็ได้) | **จำคุก ≤ 10 ปี + ปรับ ≤ 200,000** |
| **ม.12(2)** | กระทบ **ความมั่นคงของประเทศ / ความปลอดภัยสาธารณะ / เศรษฐกิจของประเทศ / โครงสร้างพื้นฐานสาธารณะ (CII)** | **จำคุก 3–15 ปี + ปรับ 60,000–300,000** |
| **ม.12 วรรคท้าย** | ทำผิดตาม **(2) แล้วเป็นเหตุให้ผู้อื่นถึงแก่ความตาย** | **จำคุก 5–20 ปี + ปรับ 100,000–400,000** ⬅️ โทษสูงสุดของทั้ง พ.ร.บ. |

**🔑 จุดสำคัญ:**
- ไล่ความรุนแรงเป็นขั้น: เสียหายต่อประชาชน (≤10 ปี) → กระทบมั่นคง/CII (3–15 ปี) → ถึงตาย (5–20 ปี)
- ชั้น (2) และวรรคท้ายมี **โทษขั้นต่ำ** (3 ปี / 5 ปี) ต่างจากมาตราอื่นที่ระบุแค่ "ไม่เกิน" → กฎหมายมองว่าร้ายแรงมาก
- "โครงสร้างพื้นฐานสาธารณะ" ใน ม.12(2) = แนวคิดเดียวกับ **CII (ม.49 พ.ร.บ.ไซเบอร์)** เช่น ไฟฟ้า ประปา การเงิน สาธารณสุข คมนาคม

**📝 ตัวอย่างเทียบ (ข้อสอบชอบออก):**
- แฮกแก้ไขเว็บเอกชนทั่วไป → **ม.9** (≤5 ปี)
- DoS ถล่มจน**ระบบจ่ายไฟ/โรงพยาบาลล่ม** → **ม.12(2)** (3–15 ปี)
- เหตุนั้นทำให้**คนไข้ ICU เสียชีวิต** → **ม.12 วรรคท้าย** (5–20 ปี)

## 📕 เจาะ ม.14 — นำข้อมูลเท็จ/ผิดกฎหมายเข้าระบบ (5 อนุมาตรา)

> โทษหลัก: **จำคุก ≤ 5 ปี / ปรับ ≤ 100,000 / หรือทั้งจำทั้งปรับ**

| อนุ | การกระทำ | คำสำคัญ |
|---|---|---|
| **(1)** | **โดยทุจริต/หลอกลวง** นำเข้าข้อมูล **เท็จ/บิดเบือน/ปลอม** ที่น่าเสียหายแก่ประชาชน | **"อันมิใช่หมิ่นประมาทตามประมวลอาญา"** ⬅️ จุดแก้ปี 2560 |
| **(2)** | นำเข้าข้อมูล **เท็จ** กระทบ **ความมั่นคง/ความปลอดภัยสาธารณะ/เศรษฐกิจ/โครงสร้างพื้นฐาน** หรือทำ **ประชาชนตื่นตระหนก** | ข่าวปลอมสร้างแพนิก |
| **(3)** | นำเข้าข้อมูลอันเป็นความผิดเกี่ยวกับ **ความมั่นคงแห่งราชอาณาจักร / ก่อการร้าย** | โยงประมวลอาญา |
| **(4)** | นำเข้าข้อมูลลักษณะ **ลามก** ที่ **ประชาชนทั่วไปเข้าถึงได้** | ต้อง "เข้าถึงได้ทั่วไป" |
| **(5)** | **เผยแพร่/ส่งต่อ (แชร์)** ข้อมูลตาม (1)–(4) **โดยรู้อยู่แล้ว** | ⭐ แชร์ก็ผิด! |

**🔑 จุดเน้น:**

1. **การแก้ปี 2560 — กัน ม.14 ไม่ให้ใช้ฟ้องหมิ่นประมาท**
   > เดิม (2550) ม.14(1) ถูกนำไปฟ้องคดีหมิ่นประมาทออนไลน์เกลื่อน → ปี 2560 เติม **"โดยทุจริต/หลอกลวง"** + **"อันมิใช่การหมิ่นประมาทตามประมวลอาญา"**
   > ⚠️ **กับดัก:** ด่า/ใส่ร้ายรายบุคคลออนไลน์ = **หมิ่นประมาท (ประมวลอาญา ม.326/328)** ไม่ใช่ ม.14 — ม.14 ใช้กับ **ข้อมูลเท็จที่หลอกลวงเป็นวงกว้าง** (ข่าวปลอม/สแกม)

2. **อนุ (5) แชร์/ส่งต่อโดยรู้ = ผิดเท่าคนโพสต์** (ไม่ใช่แค่คนต้นทาง)

3. **วรรคสอง — กรณีทำต่อ "บุคคลคนเดียว" (ไม่ใช่ประชาชน):**

| กรณี | โทษ | ยอมความ |
|---|---|---|
| ทำผิด (1) ต่อ **ประชาชนทั่วไป** | ≤ 5 ปี / ≤ 100,000 | ❌ ยอมความไม่ได้ |
| ทำผิด (1) ต่อ **บุคคลใดบุคคลหนึ่ง** | **≤ 3 ปี / ≤ 60,000** | ✅ **ยอมความได้** |

**📝 ตัวอย่างเทียบ (จำแนกให้ถูกอนุ):**
- ข่าวปลอม "ธนาคารจะล้ม รีบถอนเงิน" จนคนแห่ถอน → **ม.14(2)** (ตื่นตระหนก/เศรษฐกิจ)
- เว็บปลอมหลอกให้โอนเงิน (สแกม) → **ม.14(1)** (ทุจริต/หลอกลวง)
- โพสต์คลิปลามกให้คนทั่วไปดูได้ → **ม.14(4)**
- แชร์ข่าวปลอมต่อทั้งที่รู้ว่าปลอม → **ม.14(5)**
- โพสต์ด่า "นาย ก. โกงเงิน" (ใส่ร้ายรายบุคคล) → **หมิ่นประมาท** ไม่ใช่ ม.14

## ⚖️ กล่องเทียบ: ดูหมิ่น vs หมิ่นประมาท vs ม.14 (กันพลาดข้อสอบ)

> ทั้ง "ดูหมิ่น" และ "หมิ่นประมาท" อยู่ใน **ประมวลกฎหมายอาญา** (ไม่ใช่ พ.ร.บ.คอม)

| ประเด็น | **ดูหมิ่น** | **หมิ่นประมาท** | **ม.14 พ.ร.บ.คอม** |
|---|---|---|---|
| **มาตรา** | อาญา **ม.393** | อาญา **ม.326 / ม.328 (โฆษณา)** | พ.ร.บ.คอม **ม.14** |
| **ลักษณะ** | คำ **ด่า/เหยียด** ไม่มีเนื้อเรื่อง | **"ใส่ความ"** เล่าเรื่องที่ทำเสียชื่อเสียง | **ข้อมูลเท็จ/หลอกลวง** เป็นวงกว้าง |
| **บุคคลที่สาม** | ❌ ไม่ต้อง (ซึ่งหน้าก็ผิด) | ✅ ต้องมี | — (กระทบประชาชน) |
| **โทษ** | ≤ 1 เดือน / ≤ 10,000 | ม.326 ≤1 ปี · **ม.328 ≤2 ปี/≤200,000** | ≤ 5 ปี / ≤ 100,000 |
| **พิสูจน์ว่าจริงแล้วรอด** | ❌ | ✅ ม.330 (จริง+ประโยชน์สาธารณะ) | — |

**🔑 วิธีจำ:**
- **ดูหมิ่น = "ด่า"** (ไอ้โง่/อีเลว — พิสูจน์จริง-เท็จไม่ได้)
- **หมิ่นประมาท = "ใส่ความ"** (นาย ก. ยักยอกเงิน — พิสูจน์ได้)
- **ม.14 = "ข่าวปลอม/สแกม"** หลอกประชาชนเป็นวงกว้าง

> 💡 **ทดสอบเร็ว:** ตัดประโยคออกมาแล้ว "พิสูจน์ว่าจริงหรือเท็จได้ไหม" → ได้ = **หมิ่นประมาท** / ไม่ได้ (แค่คำด่า) = **ดูหมิ่น**
> ⚠️ **ด่า/ใส่ร้ายรายบุคคลทางออนไลน์ = ม.328 (หมิ่นประมาทโดยโฆษณา) ไม่ใช่ ม.14** — เพราะปี 2560 แก้ว่า ม.14 "อันมิใช่การหมิ่นประมาท"

## 🖼️ เจาะ ม.16 — ภาพตัดต่อทำให้เสียหาย (รวม deepfake)

> **โทษหลัก (วรรคหนึ่ง): จำคุก ≤ 3 ปี *และ* ปรับ ≤ 200,000** — สังเกตคำว่า **"และ"** (ลงโทษได้ทั้งจำทั้งปรับพร้อมกัน)

**องค์ประกอบความผิด (ต้องครบทุกข้อ):**

| # | องค์ประกอบ | จุดต้องจับ |
|---|---|---|
| 1 | นำเข้าสู่ระบบที่ **ประชาชนทั่วไปเข้าถึงได้** | โพสต์สาธารณะ (ไม่ใช่ส่งส่วนตัวคนเดียว) |
| 2 | เป็น **ภาพของผู้อื่น** | ต้องเป็น "ภาพ" |
| 3 | ภาพนั้น **สร้างขึ้น/ตัดต่อ/เติม/ดัดแปลง** | ⭐ หัวใจ — ภาพที่ถูกทำขึ้น ไม่ใช่ภาพจริง |
| 4 | น่าจะทำให้ **เสียชื่อเสียง/ถูกดูหมิ่น/ถูกเกลียดชัง/อับอาย** | องค์ประกอบเดียวกับหมิ่นประมาท |

**🔑 จุดออกสอบ:**
1. **วรรคสอง — ภาพของ "ผู้ตาย"** ที่น่าจะทำให้ **บิดา/มารดา/คู่สมรส/บุตร** เสียชื่อเสียง/อับอาย → ผิด โทษเท่าวรรคหนึ่ง (ครอบครัวเป็นผู้เสียหายแทน)
2. **วรรคสาม — ข้อยกเว้น:** ทำโดย **สุจริต เป็นการติชมด้วยความเป็นธรรม** ตามวิสัยประชาชน → **ไม่มีความผิด** (เช่น ภาพล้อ/เสียดสีเชิงวิจารณ์โดยสุจริต)
3. **เป็นความผิดอันยอมความได้** (ผู้เสียหายตายก่อนร้องทุกข์ → ครอบครัวร้องทุกข์แทนได้)

**⚠️ กับดัก — แยก ม.16 จากมาตราใกล้เคียง:**

| สถานการณ์ | มาตรา |
|---|---|
| เอา **ภาพตัดต่อ** ทำให้อับอาย | **ม.16** ✅ |
| เอา **ภาพจริง** (ไม่ตัดต่อ) มาประจาน | ❌ ไม่ใช่ ม.16 → **หมิ่นประมาท ม.328** |
| ภาพ/คลิป **ลามก** เข้าถึงทั่วไป | **ม.14(4)** |
| **ด่าด้วยข้อความ** ใส่ร้าย | **หมิ่นประมาท ม.328** |

> 💡 **คีย์เวิร์ดชี้ ม.16 = "ตัดต่อ/ดัดแปลงภาพ"** ถ้าไม่มีการตัดต่อภาพ → ไม่ใช่ ม.16
> 🆕 **deepfake** (สร้างใบหน้า/วิดีโอปลอม) ทำให้เสียชื่อเสียง = เข้า ม.16 เต็ม ๆ

**📝 ตัวอย่าง:**
- ตัดต่อหัวเพื่อนไปต่อภาพโป๊แล้วโพสต์ → **ม.16** (+ อาจ ม.14(4) ถ้าลามก)
- เอารูปจริงตอนเมาของเพื่อนมาประจาน → **ม.328** (ภาพจริง ไม่ใช่ ม.16)
- ทำมีมล้อนักการเมืองเชิงวิจารณ์โดยสุจริต → **ยกเว้นวรรคสาม ไม่ผิด**

> 💡 **ม.14 (ข่าวปลอม/หมิ่นประมาทออนไลน์)** และ **ม.15 (ผู้ให้บริการ)** เป็นมาตราที่ออกสอบบ่อยที่สุด
> **ม.15:** ถ้าผู้ให้บริการได้รับแจ้งเตือนแล้ว **นำข้อมูลออก (notice & takedown)** → **ไม่ต้องรับโทษ**
> **การ "แชร์/ส่งต่อ"** ข้อมูลผิด ม.14 โดยรู้อยู่แล้ว = มีความผิดด้วย

## 🌏 เจาะ ม.17 — ลงโทษการกระทำผิด "นอกราชอาณาจักร" (Extraterritorial Jurisdiction) ⭐ออกสอบแน่

> **เหตุผล:** อาชญากรรมคอมพิวเตอร์ **ไร้พรมแดน** — แฮกเกอร์อยู่ต่างประเทศแต่โจมตีเหยื่อในไทยได้ กฎหมายจึงตามไปลงโทษในไทยได้

**เงื่อนไข 2 กรณี (ต้องมี "คำร้องขอให้ลงโทษ" เสมอ):**

| กรณี | ผู้กระทำ | เงื่อนไขเพิ่ม |
|---|---|---|
| **ม.17(1)** | **คนไทย** | **รัฐที่เกิดเหตุ** หรือ **ผู้เสียหาย** ร้องขอให้ลงโทษ |
| **ม.17(2)** | **คนต่างด้าว** | **รัฐบาลไทย หรือ คนไทย เป็นผู้เสียหาย** + ผู้เสียหายร้องขอให้ลงโทษ |

> 🔑 **หัวใจ:**
> - ทั้ง 2 กรณี **ต้องมีการ "ร้องขอให้ลงโทษ"** เสมอ (ไม่ร้องขอ = ลงโทษในไทยไม่ได้)
> - กรณี (1) ดูที่ **สัญชาติผู้กระทำ (คนไทย)** / กรณี (2) ดูที่ **ผู้เสียหายเป็นไทย/รัฐไทย**
> - หลักกฎหมาย: **personality principle** (สัญชาติคนไทย) + **protective/passive principle** (คุ้มครองคนไทย/รัฐไทย)

## อำนาจพนักงานเจ้าหน้าที่ & เขตอำนาจ
- **ม.17/1** — คณะกรรมการ **เปรียบเทียบปรับ** คดีเล็กน้อย (ชำระแล้วคดีอาญาเลิกกัน)
- **ม.18(1)–(3)** — อำนาจ **ไม่ต้องขอหมายศาล**: เรียกบุคคลให้ถ้อยคำ/ส่งเอกสาร, เรียกข้อมูลจราจร, สั่งผู้ให้บริการส่งข้อมูล (ผู้รับคำสั่งต้องทำใน 7–15 วัน)
- **ม.18(4)–(8) + ม.19** — อำนาจ **ต้องขอศาล**: ทำสำเนา/สั่งส่งมอบ/เข้าถึงระบบ/ถอดรหัส/ยึด-อายัด; รายงานศาลภายใน **48 ชั่วโมง**; ยึด/อายัด ≤ 30 วัน (ขยายสูงสุด 60 วัน)
- **ม.20** — การ **ระงับ/ลบข้อมูล** ที่ผิดกฎหมายชัดเจนหรือขัดศีลธรรม ต้องผ่านความเห็นชอบ + **ยื่นคำร้องต่อศาล**
- **ม.21** — ห้ามจำหน่าย/เผยแพร่ **ชุดคำสั่งไม่พึงประสงค์ (มัลแวร์)**
- **ม.22–24 (ความรับผิดของเจ้าหน้าที่เอง)** — เปิดเผยข้อมูลโดยเจตนา (ม.22) จำคุก ≤3 ปี/ปรับ ≤60,000 · ประมาทเลินเล่อ (ม.23) ≤1 ปี/≤20,000 · บุคคลภายนอกรู้แล้วเปิดเผย (ม.24) ≤2 ปี/≤40,000
- **ม.25** — การรับฟัง **พยานหลักฐานดิจิทัล** (ที่ได้มาโดยมิชอบอาจรับฟังไม่ได้)
- **ม.26** — ผู้ให้บริการต้องเก็บ **ข้อมูลจราจร (log) ไม่น้อยกว่า 90 วัน** (สั่งขยายได้สูงสุด 2 ปี); ฝ่าฝืน **ปรับไม่เกิน 500,000 บาท**
- **ม.27** — ขัดคำสั่งศาล/พนักงานเจ้าหน้าที่ → **ปรับไม่เกิน 200,000 บาท + ปรับรายวัน ≤ 5,000 บาท/วัน**

# ส่วน B — พ.ร.บ.การรักษาความมั่นคงปลอดภัยไซเบอร์ พ.ศ. 2562

| รายการ | ข้อมูล |
|---|---|
| ประกาศ / บังคับใช้ | 27 พ.ค. 2562 / **28 พ.ค. 2562** |
| แนวคิด | เปลี่ยน "ความปลอดภัยไซเบอร์" จากเรื่องเทคนิค → เป็น **หน้าที่ตามกฎหมาย** ของหน่วยงานและผู้บริหาร |

## ⭐ นิยามสำคัญ (3 ตัว)
- **ภัยคุกคามทางไซเบอร์ (มาตรา 3)** — การกระทำที่มิชอบต่อระบบ/ข้อมูล ที่อาจก่อความเสียหาย
- **CII — โครงสร้างพื้นฐานสำคัญทางสารสนเทศ (มาตรา 49)** — ระบบของบริการสำคัญ เช่น ไฟฟ้า ประปา การเงิน สาธารณสุข คมนาคม
- **ระดับความรุนแรงของภัย (มาตรา 60) — 3 ระดับ:** ① ไม่ร้ายแรง ② ร้ายแรง ③ วิกฤติ

## ⭐ โครงสร้างองค์กร (The National Ecosystem) — จำลำดับ
**กมช. (นโยบาย ม.5) → กกท. (กำกับดูแล ม.12) → สกมช. (สำนักงาน=ปฏิบัติ) → หน่วยงาน CII (ปฏิบัติจริง)**
- **กมช.** = คณะกรรมการการรักษาความมั่นคงปลอดภัยไซเบอร์แห่งชาติ (นายกฯ เป็นประธาน) — กำหนดนโยบายระดับชาติ
- **กกท.** = คณะกรรมการกำกับดูแลด้านความมั่นคงปลอดภัยไซเบอร์
- **สกมช.** = สำนักงานคณะกรรมการการรักษาความมั่นคงปลอดภัยไซเบอร์แห่งชาติ (หน่วยปฏิบัติ)

## หน้าที่และการรับมือ
- **มุมกฎหมาย:** หน้าที่และความรับผิด (ม.44–46) — หน่วยงาน CII ต้องมีมาตรการ ประเมินความเสี่ยง แจ้งเหตุ
- **มุมเทคนิค:** Log · Monitoring · Incident Response · Backup · Access Control
- **กระบวนการรับมือเหตุ (Incident Response):** ตรวจพบ → ตรวจสอบ → ประเมิน → รับมือ → รายงาน
- **วงจรความมั่นคงไซเบอร์ (Cycle of Resilience):** Identify → Protect → Detect → Respond → Recover
- **โทษ (ม.70–76)** เช่น เข้าถึง CII โดยมิชอบ จำคุก ≤ 2 ปี/ปรับ ≤ 60,000
- **ความรับผิดของผู้บริหาร (ม.77)** — หากนิติบุคคลทำผิดจากการสั่งการ/ละเว้นของผู้บริหาร ผู้บริหารต้องรับผิดด้วย (Executive Liability)

---
## 🎯 มิติจริยธรรม (สำหรับอัตนัย)
- **พ.ร.บ.คอม** ตั้งอยู่บนหลัก: เคารพสิทธิผู้อื่น · เคารพความเป็นส่วนตัว · รักษาความถูกต้องของข้อมูล (Integrity) · ใช้เสรีภาพอย่างรับผิดชอบ · ไม่ก่อความเสียหาย (Non-Maleficence)
- **พ.ร.บ.ไซเบอร์** เพิ่ม: ความมั่นคงของส่วนรวม (Collective Security) · บริหารความเสี่ยงร่วมกัน · ใช้อำนาจรัฐอย่างได้สัดส่วน (Proportionality) · ความโปร่งใสตรวจสอบได้
- ประโยคสรุป: *"ใช้เทคโนโลยีต้องเคารพสิทธิผู้อื่น รับผิดชอบต่อสังคม รักษาความมั่นคงของข้อมูล/ระบบ และคำนึงถึงประโยชน์ส่วนรวมควบคู่กับการคุ้มครองสิทธิเสรีภาพ"*
